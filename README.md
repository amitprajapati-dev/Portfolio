# Amit Portfolio

A modern developer portfolio built with Next.js and Tailwind CSS.

## About

This portfolio showcases my projects, skills, and journey as a software developer.

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- Git & GitHub

## Features

- Responsive Design
- Modern UI/UX
- Smooth Animations
- Project Showcase
- Skills Section
- Contact Section

## 📦 Installation

```bash
git clone <repository-url>
cd portfolio
npm install
npm run dev
```
